# Christ_Days_Calendar_Reminder
Prepared for server-event reminders and devotional room use; a piece of coding that automatically marks detailed calendar dates for Holy Days related to Christ with an 8-day reminder before the event plus useful information. [Has Nice Banners In Package]

CHRIST DAYS CALENDAR REMINDER BUILDER
Version 0.1.0 — 25 May 2026
Signature: NemesisC64 × Lexmilian de Mello / Percarus

WHAT THIS PACKAGE DOES
- Opens a small polished Windows-style desktop app.
- Reads Jesus_Centred_Christian_Holiday_Guide_2026.txt.
- Uses the images in the folder: High Res Banners for Days of Christ
- Shows each event and banner in the app.
- Generates ADD .ics calendar files for Google Calendar, Microsoft/Outlook, Apple Calendar, and Linux calendar apps.
- Adds an 8-day display reminder alarm by default.
- Calculates moving Christian dates such as Ash Wednesday, Palm Sunday, Easter, Ascension, Pentecost, Corpus Christi, Sacred Heart, and Christ the King for the chosen year range.
- Generates a matching REMOVE/CANCEL .ics file using the exact UIDs created by the app.

IMPORTANT LIMITATION
This is not a signed precompiled Windows .exe yet, because this package was generated outside a Windows build environment.
Run Build_EXE.bat or Build_EXE.ps1 on Windows to create the executable.

HOW TO RUN WITHOUT BUILDING
1. Install Python 3.11+ on Windows.
2. Open this folder.
3. Double-click Christ_Days_Calendar_Reminder.py if Python/Tkinter is associated, or run:
   python Christ_Days_Calendar_Reminder.py

HOW TO BUILD THE .EXE
1. Install Python 3.11+ and tick "Add Python to PATH".
2. Double-click Build_EXE.bat.
3. The created executable will be:
   dist\Christ_Days_Calendar_Reminder.exe
4. Keep the EXE near:
   - Jesus_Centred_Christian_Holiday_Guide_2026.txt
   - High Res Banners for Days of Christ\

CALENDAR SAFETY
The app creates event UIDs like:
percarus-christ-days-XX-event-name-YYYY@percarus-christ-days.local

That means the REMOVE/CANCEL file is designed to target only events generated by this app, not unrelated calendar events on the same holiday dates.

Best practice:
Create a separate calendar named "Christ Days Reminders" and import the ADD file there. Then, if you ever want to remove everything at once, delete that separate calendar or import the REMOVE/CANCEL file.

LIVE LOGIN / API FUTURE WORK
Proper Google or Microsoft login must use OAuth. It requires developer credentials registered to your own Google Cloud or Microsoft Entra/Azure app. This package intentionally does not collect usernames or passwords.

Apple Calendar and many Linux calendar apps normally work best with .ics import rather than a universal login button.
